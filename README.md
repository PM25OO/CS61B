## 📚 课程内容

### 🧪 Labs（实验）
- **lab1** - 入门与环境配置
- **lab2** - 单元测试与调试
- **lab3** - 随机化测试与时间复杂度
- **lab4** - Git进阶与继承
- **lab5** - 同行代码审查
- **lab6** - Git工作流
- **lab7** - 二叉搜索树
- **lab8** - 哈希表

### 🚀 Projects（项目）
- **proj0** - 2048游戏
- **proj1** - 双端队列（Deque）
- **proj1ec** - 额外测试
- **proj2** - Gitlet（版本控制系统）
- **proj3** - 构建你自己的世界（BYOW）
